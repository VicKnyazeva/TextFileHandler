Описание работы приложения "Text File Processor"

1. Для обработки текста пользователю необходимо выбрать текстовый файл с расширением .txt нажатием на кнопку "Выбрать файл для обработки". Будет открыто диалоговое окно, где можно выбрать обрабатываемый файл.

1.1. Примечание: Приложение обрабатывает файлы только с расширением .txt и только с кодировкой UTF-8.

2. Для сохранения результатов обработки текста пользователю необходимо указать файл нажатием на кнопку "Сохранить результат обработки". Будет открыто диалоговое окно, где пользователь может выбрать или создать текстовый файл. По умолчанию новый текстовый файл имеет имя "NewDocument.txt".

3. Пользователь может указать настройки для обработки текста:
3.1. Удалять знаки препинания. При включении этой опции все знаки препинания в обрабатываемом тексте будут удалены.
3.2. Удалять пустые строки. Эта опция позволяет оптимизировать итоговый текст, исключив в нем пустые строки.
3.3. Исключить слова меньше, чем (количество символов). Данная опция позволяет исключить все слова, менее указанно числа символов.
3.4. По умолчанию удаление знаков препинания и пустых строк не включено, а минимальное количество символов установлено в 0. 
При таком наборе опций в результате обработки пользователь получит файл идентичный исходному.

4. Обработка текста запускается нажатием на кнопку "Обработать файл". 
По завершении обработки пользователь получит сообщение в модальном диалоге.


Необходимые программные средства для сборки и запуска приложения "Text File Processor"

1. Для открытия решения и последующей сборки приложения необходима Microsoft Visual Studio.NET. 
Достаточно версии Community 2019.
2. Установленный Microsoft .NET Framework Version 4.7.2
